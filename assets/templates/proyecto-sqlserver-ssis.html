Proyecto SQL Server y SSIS

Este proyecto mayormente hace referencia a SQL Server Integration Services, procesos ETL que permiten realizar un procesamiento de datos para luego de procesarlos y manipularlos puedan ser ingresados en un determinado destino , pdiendo ser en este caso Bases de datos Relacionales de SQL Server.

SQL Server Query

Tambien tengo conocimientos en Sentencias SQL pudiendo dividirlas en dos Grupos

  • DDLs
  • DMLs

Dentro de las DDLs podemos encontrar los tipos de Sentencias

  • Create
    CREATE TABLE tabla (
    id INT identity(1,1),
    columna1 VARCHAR(25) not null,
    columna2 DECIMAL(10,2) not null,
    columna3 INT null
    )
  • Alter
    ALTER TABLE ALTER columna1 VARCHAR(25) null
  • Drop
    DROP TABLE tabla

Dentro de las DMLs podemos encontrar los tipos de Sentencias

  • Select
  • Update
  • Insert
  • Delete

Los Update , Insert y Delete son querys unicas que siguen un patron sin embarg la unica que tiene una gran cantidad de variaciones es la select ya que es bastante variable por su cantidad de funciones y variaciones que se pueden usar
