APP de Testimonios

Estos testimonios fueron creados a traves de un componente general que se programo con props para que fuera dinamico por cada testimonio


APP de Contador de Click.

Esta app de contador de clicks fue diseñada usando dos componentes, uno para los botones y otro para el cuadro del contador, tambien se adiciono el uso de HOOKs mara manipular los estados.


APP de Calculadora

Esta app de calculadora fue diseñada usando tres componentes, de igual manera se usaron Hooks y tambien se agregaron algunas librerias como por ejemplo Mathjs para usar algunas funcionalidades de resolver calculos aritmeticos con String


APP de Tareas

Esta app de Tareas fue diseñada usando tres componentes, de igual manera se usaron Hooks y tambien se agregaron algunas librerias como por ejemplo uuid para usar algunas funcionalidades de control de id, esta app tiene una variedad de funcionalidades que van desde : Agregar tareas, Eliminar Tareas, marcarlas como listas entre otras.
